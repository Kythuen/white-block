## Installation

::: code-group
```sh [pnpm]
pnpm add @white-block/vitepress
```
```sh [npm]
npm install @white-block/vitepress
```
```sh [yarn]
yarn add @white-block/vitepress
```
:::

## Usage

### CLI

Use CLI tool provided, you can quick create a vitepress project with WhiteBlock theme.

::: code-group
```sh [pnpm]
npx wb-vitepress init
```
```sh [npm]
npx wb-vitepress init
```
```sh [yarn]
wb-vitepress init
```
:::
